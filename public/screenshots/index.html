PWA Screenshot Templates

This directory contains templates for the screenshots used in the PWA manifest. These screenshots enhance the installation experience of the PWA by providing visual previews in the installation prompt.

Available Templates

  • desktop-home.html/png - Desktop view of the homepage (1280x720)
  • desktop-projects.html/png - Desktop view of the projects section (1280x720)
  • mobile-home.html/png - Mobile view of the homepage (750x1334)

Note: The HTML files serve as templates that can be converted to PNG images using the provided conversion script. The actual PNG files are referenced in the PWA manifest.

How to Use

To convert the HTML templates to PNG screenshots:

  1. Make sure you have either wkhtmltoimage, Chrome/Chromium, or ImageMagick installed
  2. Run the conversion script: bash convert-screenshots.sh
  3. The generated PNG files will be used by the PWA manifest

Alternatively, you can open the HTML files in a browser and take screenshots manually.

Customization

Feel free to modify the HTML templates to better match your portfolio's design:

  • Change colors, fonts, and layout in the CSS
  • Update content to reflect your actual portfolio information
  • Add your logo or personal branding elements

After making changes, re-run the conversion script to update the PNG files.
